
# Privacy Leak Alert App

## Overview
Privacy Leak Alert App is designed to detect potential privacy leaks in user data such as phone numbers, email addresses, and sensitive information before sharing it online.

## Features
- Detect email addresses
- Detect phone numbers
- Alert users about possible privacy leaks
- Secure data handling

## Technologies Used
- Flutter (Frontend)
- Python / FastAPI (Backend)
- GitHub for version control
